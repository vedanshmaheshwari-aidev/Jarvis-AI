# Jarvis AI OS

> A fully local AI operating assistant powered by Ollama.

## Vision

Jarvis AI OS is a production-quality desktop AI assistant capable of:

- 💬 Natural conversation
- 📂 Local file management
- 💻 Coding assistance
- 📊 Data analysis
- 🧠 Long-term memory
- 🤖 Multi-step task execution

## Tech Stack

- Python 3.12
- Ollama
- Qwen3
- PySide6
- SQLite
- ChromaDB
